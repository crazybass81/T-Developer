## 📋 작업 구조 체계
```yaml
Phase → Task → Subtask → Unit
- Phase: 주요 단계 (6개 Phase)
- Task: 핵심 작업 (Phase당 4-5개)
- Subtask: 세부 작업 (Task당 3-4개)
- Unit: 실행 단위 (Subtask당 2-3개)
```
