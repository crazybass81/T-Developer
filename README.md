# T-Developer MVP

AI-powered multi-agent development platform.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+
- AWS Account
- Docker

### Installation
```bash
# Check requirements
bash scripts/check-requirements.sh

# Install dependencies (coming next)
npm install
```
